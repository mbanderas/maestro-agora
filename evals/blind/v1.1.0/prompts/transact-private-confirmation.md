# Request

Write a confirmation message for a signed-in project dashboard after a user archives a client report. Include a short title, one sentence of body copy, and a button label that returns to the report list.

## Supplied facts

- The selected report has been archived successfully.
- Archived reports no longer appear in the active report list.
- The user can restore an archived report later from Archive.
- The report data was not deleted.
- The report-list route exists.

## Constraints

- State what happened and what the user can do next.
- Do not manufacture risk, urgency, or a sales message.
- Use a specific accessible button label.
