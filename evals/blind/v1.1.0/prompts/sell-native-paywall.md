# Request

Write a native-app paywall with a headline, body copy, and one button label for a Briefcase Free user who tries to create another active client report.

## Supplied facts

- Briefcase Free permits three active client reports.
- All three slots are currently in use.
- Briefcase Pro removes the active-report limit and includes PDF export.
- A working upgrade route exists.
- No savings, discount, deadline, or urgency claim is available.

## Constraints

- Make the current state and next available action clear.
- Do not use fake urgency or hide the plan difference.
- Keep the interface copy compact and accessible.
